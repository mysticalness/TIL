
# revert, reset



## 작업 되돌리기
  * 만들어진 버전을 되돌리는 두 가지 방법
    * revert : 버전을 되돌린 새로운 버전 만들기
    * reset : 버전을 완전히 되돌리기

## reset의 종류

| 종류 | 내용 |
| ---- | --- |
| soft reset |	커밋만 되돌리기 |
| mixed reset |	스테이지까지 되돌리기 |
| hard reset |	작업 디렉터리까지 되돌리기 |

## 하나의 버전이 만들어지는 과정 (reset 적용)
  * ~~작업 디렉터리에서 변경 사항 생성하기~~   `hard reset`
  * ~~스테이지로 추가하기~~   `mixed reset`
  * ~~저장소로 커밋하기~~   `soft reset`
